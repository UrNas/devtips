# Navigate Terminal Like a Pro

**Category:** `terminal`  ·  **Format:** `shorts`

**Tags:** terminal, shortcuts, navigation, productivity, cli, keyboard, efficiency

## Steps

### 1. Jump anywhere instantly

```bash
$ cd ~/Documents/project
```

### 2. Ctrl+A jumps to line start

```bash
$ echo 'very long command here'
very long command here
```

### 3. Ctrl+E jumps to line end

```bash
$ pwd
/Users/dev/Documents/project
```

### 4. Ctrl+U clears entire line

```bash
$ ls -la
total 8
```
