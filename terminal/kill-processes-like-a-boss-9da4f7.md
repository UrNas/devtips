# Kill Processes Like a Boss

**Category:** `terminal`  ·  **Format:** `shorts`

**Tags:** terminal, processes, kill, ctrl-c, ps, killall, debugging

## Steps

### 1. Stop frozen processes instantly

```bash
$ ps aux | grep node
user  1234  0.0  1.2  node server.js
user  5678  0.0  0.8  node build.js
```

### 2. Ctrl+C kills current process

```bash
$ npm start
Server running on port 3000
^C
```

### 3. kill -9 forces termination

```bash
$ kill -9 1234
```

### 4. killall kills by name

```bash
$ killall node
$ ps aux | grep node
No matching processes found
```
