# Create Custom Terminal Shortcuts

**Category:** `terminal`  ·  **Format:** `shorts`

**Tags:** terminal, aliases, shortcuts, bashrc, customization, productivity, git

## Steps

### 1. Turn long commands into shortcuts

```bash
$ git log --oneline --graph --all
* a1b2c3d Add feature
* d4e5f6g Fix bug
```

### 2. Create alias in ~/.bashrc

```bash
$ echo "alias gl='git log --oneline --graph --all'" >> ~/.bashrc
```

### 3. Reload your shell config

```bash
$ source ~/.bashrc
```

### 4. Now 'gl' runs the full command

```bash
$ gl
* a1b2c3d Add feature
* d4e5f6g Fix bug
* h7i8j9k Initial commit
```
