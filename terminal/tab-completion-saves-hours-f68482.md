# Tab Completion Saves Hours

**Category:** `terminal`  ·  **Format:** `shorts`

**Tags:** terminal, tab-completion, autocomplete, files, commands, productivity, typing

## Steps

### 1. Stop typing full file names

```bash
$ ls
package.json
src/
node_modules/
README.md
```

### 2. Type 'pac' then hit Tab

```bash
$ cat package.json
{
  "name": "my-app",
  "version": "1.0.0"
}
```

### 3. Double Tab shows all options

```bash
$ cd s
$ ls src/
components/
utils/
app.js
index.js
```

### 4. Works with commands too

```bash
$ npm install
added 425 packages in 12s
```
