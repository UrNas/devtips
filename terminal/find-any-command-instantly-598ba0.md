# Find Any Command Instantly

**Category:** `terminal`  ·  **Format:** `shorts`

**Tags:** terminal, history, search, ctrl-r, commands, productivity, shortcuts

## Steps

### 1. Never type long commands again

```bash
$ git log --oneline --graph --all
* a1b2c3d Add feature
* d4e5f6g Fix bug
```

### 2. Ctrl+R searches command history

```bash
$ history | grep git
  142  git status
  143  git log --oneline --graph --all
```

### 3. Type 'git' then Ctrl+R

```bash
$ git status
On branch main
nothing to commit, working tree clean
```

### 4. Press Enter to run it

```bash
$ !142
On branch main
nothing to commit, working tree clean
```
