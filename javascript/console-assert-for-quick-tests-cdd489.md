# Console.assert for Quick Tests

**Category:** `javascript`  ·  **Format:** `shorts`

**Tags:** javascript, console, assert, testing, debugging, validation

## Steps

### 1. Assert validates assumptions inline

```javascript
const age = 25;
console.assert(age >= 18, 'Must be adult');
```

### 2. Only logs when assertion fails

```javascript
const score = 150;
console.assert(score <= 100, 'Invalid score:', score);
```

### 3. Perfect for development checks

```javascript
function divide(a, b) {
  console.assert(b !== 0, 'Division by zero');
  return a / b;
}
```

### 4. Clean alternative to if statements

```javascript
// Instead of if (!user) throw error
console.assert(user, 'User required');
```
