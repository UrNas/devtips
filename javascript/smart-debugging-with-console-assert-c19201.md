# Smart Debugging with Console Assert

**Category:** `javascript`  ·  **Format:** `shorts`

**Tags:** javascript, debugging, assertions, console, conditional, testing

## Steps

### 1. Only log when conditions fail

```javascript
const user = { age: 16 };
console.assert(user.age >= 18, 
'User must be 18+');
```

### 2. Passing assertions stay silent

```javascript
const admin = { age: 25 };
console.assert(admin.age >= 18, 
'Admin must be 18+');
```

### 3. Add helpful context data

```javascript
console.assert(user.email, 
'Missing email', { user, 
timestamp: Date.now() });
```
