# Style Your Console Logs with CSS

**Category:** `javascript`  ·  **Format:** `shorts`

**Tags:** javascript, console, css, styling, debugging, formatting

## Steps

### 1. Add colors to console output

```javascript
console.log('%cSuccess!', 
'color: green; font-weight: bold');
```

### 2. Multiple styles in one log

```javascript
console.log('%cError: %cFile not found',
'color: red', 'color: orange');
```

### 3. Add background and padding

```javascript
console.log('%cAPI Response', 
'background: #007acc; color: white; 
padding: 4px 8px');
```
