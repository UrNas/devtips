# Debug Variables with console.dir()

**Category:** `javascript`  ·  **Format:** `shorts`

**Tags:** javascript, console, debugging, objects, console.dir, development, tips

## Steps

### 1. See object properties clearly

```javascript
const user = {
  name: 'Alice',
  skills: ['JS', 'React']
};
```

### 2. console.log shows basic output

```javascript
console.log(user);
```

### 3. console.dir reveals everything

```javascript
console.dir(user);
```

### 4. Perfect for DOM elements too

```javascript
console.dir(document.body);
```
