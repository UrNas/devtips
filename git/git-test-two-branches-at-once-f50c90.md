# Git: Test Two Branches at Once

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, worktree, testing, branches, parallel, development, workflow

## Steps

### 1. Create worktree for branch comparison

```bash
$ git worktree add ../feature feature/auth
Preparing worktree (new branch 'feature/auth')
HEAD is now at abc1234 Add auth
```

### 2. Run tests in main directory

```bash
$ npm test
✓ All tests passing (12/12)
Duration: 2.3s
```

### 3. Switch to worktree & test there

```bash
$ cd ../feature && npm test
✓ All tests passing (14/14)
Duration: 2.1s
```

### 4. Both branches tested simultaneously!

```bash
$ git worktree list
/project     abc1234 [main]
/feature     def5678 [feature/auth]
```
