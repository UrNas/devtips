# Git: Watch Tests on Live Branches

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, worktree, watch, live, testing, development, workflow

## Steps

### 1. Set up main branch watcher

```bash
$ git worktree add ../main-watch main
Preparing worktree
HEAD is now at abc1234
```

### 2. Start file watcher with tests

```bash
$ cd ../main-watch && npm run test:watch &
[1] 12345
Watching for file changes...
```

### 3. Watch current branch too

```bash
$ npm run test:watch &
[2] 12346
Watching for file changes...
```

### 4. Two live test runners active!

```bash
$ jobs
[1]   Running   npm run test:watch
[2]+  Running   npm run test:watch
```
