# Git: Auto-Test Feature Branches

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, automation, worktree, bash, parallel, testing, ci

## Steps

### 1. Create test script for automation

_test-all.sh_

```bash
#!/bin/bash
for branch in main feature/*; do
  git worktree add ../$branch $branch
done
```

### 2. Add parallel test execution

_test-all.sh_

```bash
for dir in ../*/; do
  (cd $dir && npm test) &
done
wait
```

### 3. Make script executable

```bash
$ chmod +x test-all.sh
```

### 4. Test all branches at once!

```bash
$ ./test-all.sh
Testing main... ✓
Testing feature/auth... ✓
Testing feature/ui... ✓
```
