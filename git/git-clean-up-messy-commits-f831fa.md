# Git: Clean Up Messy Commits

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, interactive-rebase, commit-history, squash, reword, clean-commits

## Steps

### 1. Interactive rebase fixes commit chaos

```bash
$ git rebase -i HEAD~3
Opening editor for interactive rebase...
Edit the commands for each commit:
```

### 2. Squash combines multiple commits

```bash
pick f7f3f6d Add login form
squash 310154e Fix typo
squash a5f4a0d Update styles
```

### 3. Reword changes commit messages

```bash
reword f7f3f6d Add login form
fix 310154e Fix validation bug
```

### 4. Drop removes unwanted commits

```bash
pick f7f3f6d Add login form
drop 310154e Debug print statements
pick a5f4a0d Add tests
```

### 5. Clean history ready to push

```bash
$ git log --oneline -3
b2c3d4e Add comprehensive login system
e5f6g7h Add input validation
h8i9j0k Previous commit
```
