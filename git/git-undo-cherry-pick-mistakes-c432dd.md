# Git: Undo Cherry-Pick Mistakes

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, cherry-pick, undo, git-reset, git-revert, git-mistakes, version-control, git-tips

## Steps

### 1. Cherry-picked wrong commit? Undo it

```bash
$ git log --oneline -1
7a8b9c0 Wrong commit cherry-picked
```

### 2. Reset removes the last commit

```bash
$ git reset --hard HEAD~1
HEAD is now at 5d6e7f8 Previous commit
```

### 3. Or revert if already pushed

```bash
$ git revert 7a8b9c0
[main 9e8f7d6] Revert "Wrong commit"
1 file changed, 2 deletions(-)
```

### 4. Cherry-pick mistake safely undone

```bash
$ git log --oneline -2
9e8f7d6 Revert "Wrong commit"
5d6e7f8 Previous commit
```
