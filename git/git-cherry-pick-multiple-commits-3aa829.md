# Git: Cherry-Pick Multiple Commits

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, cherry-pick, commit-range, multiple-commits, git-workflow, version-control, git-advanced, git-tips

## Steps

### 1. Cherry-pick a range of commits at once

```bash
$ git cherry-pick abc123d..def456e
[main 1a2b3c4] First commit
[main 5d6e7f8] Second commit
[main 9g0h1i2] Third commit
```

### 2. Check the commits you want first

```bash
$ git log --oneline feature~3..feature
def456e Latest feature
cba987f Middle commit
abc123d First feature
```

### 3. Range excludes the first hash

```bash
$ git cherry-pick abc123d^..def456e
[main 2b3c4d5] First feature
[main 6e7f8g9] Middle commit
[main 0h1i2j3] Latest feature
```

### 4. All three commits now on main

```bash
$ git log --oneline -3
0h1i2j3 Latest feature
6e7f8g9 Middle commit
2b3c4d5 First feature
```
