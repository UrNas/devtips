# Git: Cherry-Pick Specific Commits

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, cherry-pick, commits, branches, git-workflow, version-control, git-tips, development

## Steps

### 1. Move commits between branches easily

```bash
$ git cherry-pick abc123d
[main 7f8e9a2] Fix user authentication bug
1 file changed, 3 insertions(+), 1 deletion(-)
```

### 2. View available commits first

```bash
$ git log --oneline feature-branch
abc123d Fix user authentication bug
def456e Add new validation
789ghij Update docs
```

### 3. Pick the exact commit you need

```bash
$ git checkout main
Switched to branch 'main'
$ git cherry-pick abc123d
[main 7f8e9a2] Fix user authentication bug
```

### 4. That commit is now on main branch

```bash
$ git log --oneline -2
7f8e9a2 Fix user authentication bug
5a1b2c3 Previous commit
```
