# Git: Compare Test Results Instantly

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, worktree, diff, testing, comparison, progress, analysis

## Steps

### 1. Create worktrees for comparison

```bash
$ git worktree add ../old v1.0
Preparing worktree (detached HEAD)
HEAD is now at abc1234
```

### 2. Run tests and save results

```bash
$ npm test > current.log
Tests completed
$ cd ../old && npm test > old.log
Tests completed
```

### 3. Compare test outputs side by side

```bash
$ diff -y current.log ../old/old.log
12 passing    |    10 passing
2 failing     |    4 failing
```

### 4. See improvements at a glance!

```bash
$ echo 'Progress: +2 tests, -2 failures'
Progress: +2 tests, -2 failures
```
