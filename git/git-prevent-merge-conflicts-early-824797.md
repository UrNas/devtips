# Git: Prevent Merge Conflicts Early

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, prevent, conflicts, rebase, pull, communication, workflow

## Steps

### 1. Pull latest before starting work

```bash
$ git pull origin main
Already up to date
```

### 2. Rebase keeps history linear & clean

```bash
$ git rebase main
Successfully rebased
```

### 3. Small frequent commits reduce conflicts

```bash
# Commit often, not huge chunks
git add .
git commit -m "Add login validation"
```

### 4. Communicate changes to avoid overlap

```markdown
## PR: Working on auth system

Files I'm changing:
- src/auth.js
- src/login.js
```
