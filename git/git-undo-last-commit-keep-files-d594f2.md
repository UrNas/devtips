# Git: Undo Last Commit (Keep Files)

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, undo, reset, commit, soft, beginner, mistakes, staging

## Steps

### 1. Oh no! Wrong commit message 😱

```bash
$ git log --oneline -2
a1b2c3d Fix typo in header
d4e5f6g Add login validation
```

### 2. Reset keeps your changes

```bash
$ git reset --soft HEAD~1
```

### 3. Files stay staged, ready to recommit

```bash
$ git status
Changes to be committed:
  modified:   header.html
  modified:   styles.css
```

### 4. Now commit with the right message

```bash
$ git commit -m 'Fix header typo'
[main b7c8d9e] Fix header typo
 2 files changed, 4 insertions(+)
```
