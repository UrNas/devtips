# Git: Find Lost Commits with Reflog

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, reflog, recovery, lost commits, hard reset, history, rescue

## Steps

### 1. Hard reset deleted your commits? 💀

```bash
$ git log --oneline
d4e5f6g Add login validation
g7h8i9j Initial commit
```

### 2. Reflog shows ALL your git history

```bash
$ git reflog
d4e5f6g HEAD@{0}: reset: moving to HEAD~2
a1b2c3d HEAD@{1}: commit: Fix header bug
b7c8d9e HEAD@{2}: commit: Add navbar
```

### 3. Found it! Recover the lost commit

```bash
$ git checkout a1b2c3d
HEAD is now at a1b2c3d Fix header bug
```

### 4. Create branch to keep the recovery

```bash
$ git checkout -b recovered
Switched to a new branch 'recovered'
```
