# Git: Partial Stash Specific Files

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, stash, files, partial, workflow, selective, pathspec

## Steps

### 1. Multiple files changed but only stash some?

```bash
$ git status --short
M  src/app.js
M  src/config.js
M  src/utils.js
```

### 2. Stash only specific files with pathspec

```bash
$ git stash push -m "config" src/config.js
Saved working directory and index state
WIP on main: abc123 config
```

### 3. Check what's left unstashed

```bash
$ git status --short
M  src/app.js
M  src/utils.js
```

### 4. Use wildcards to stash multiple patterns

```bash
$ git stash push -m "utils" src/*.js
Saved working directory and index state
```

### 5. Perfect for partial commits workflow!

```bash
$ git stash list
stash@{0}: WIP on main: utils
stash@{1}: WIP on main: config
```
