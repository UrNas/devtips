# Git: Fix Cherry-Pick Conflicts

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, cherry-pick, merge-conflicts, git-conflicts, conflict-resolution, git-workflow, version-control, git-tips

## Steps

### 1. Cherry-pick hits a conflict? Fix it fast

```bash
$ git cherry-pick abc123d
error: could not apply abc123d... Fix bug
hint: after resolving conflicts, mark the corrected paths
hint: with 'git add <paths>' or 'git rm <paths>'
```

### 2. See which files have conflicts

```bash
$ git status
You are currently cherry-picking commit abc123d.
Unmerged paths:
  both modified:   src/auth.js
```

### 3. Edit file, then add and continue

```bash
$ git add src/auth.js
$ git cherry-pick --continue
[main 9d8e7f6] Fix bug
1 file changed, 2 insertions(+)
```

### 4. Cherry-pick complete with your fixes

```bash
$ git log --oneline -1
9d8e7f6 Fix bug
```
