# Git: Switch Branches With Uncommitted Work

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, checkout, switch, uncommitted, merge, workflow, changes

## Steps

### 1. Git won't let you switch with changes?

```bash
$ git checkout main
error: Your local changes would be
overwritten by checkout
```

### 2. Force switch and bring changes along

```bash
$ git checkout -m main
M       src/app.js
Switched to branch 'main'
```

### 3. Your uncommitted work moves too!

```bash
$ git status --short
M  src/app.js
```

### 4. If conflicts occur, Git will warn you

```bash
$ git checkout -m feature
warning: 1 path had merge conflicts
```

### 5. No more stashing for simple switches!

```bash
$ git branch
  main
* feature
```
