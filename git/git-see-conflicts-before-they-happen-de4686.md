# Git: See Conflicts Before They Happen

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, merge, conflicts, preview, git-status, git-merge, workflow, branching

## Steps

### 1. Preview merge conflicts before merging

```bash
$ git merge --no-commit --no-ff feature
Auto-merging src/app.js
CONFLICT (content): Merge conflict in src/app.js
Automatic merge failed; fix conflicts and commit
```

### 2. Check which files have conflicts

```bash
$ git status --porcelain
UU src/app.js
UU src/utils.js
```

### 3. UU means both sides modified the file

```bash
$ git diff --name-only --diff-filter=U
src/app.js
src/utils.js
```

### 4. Abort if conflicts look complex

```bash
$ git merge --abort
HEAD is now at a1b2c3d Latest feature work
```

### 5. Your branch is back to clean state

```bash
$ git status
On branch feature
nothing to commit, working tree clean
```
