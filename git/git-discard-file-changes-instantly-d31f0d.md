# Git: Discard File Changes Instantly

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, restore, discard, changes, files, undo, reset, beginner

## Steps

### 1. Broke your file? Don't panic! 🔥

```bash
$ git status
Changes not staged for commit:
  modified:   app.js
  modified:   styles.css
```

### 2. Restore discards all changes

```bash
$ git restore app.js
```

### 3. File is back to last commit state

```bash
$ git status
Changes not staged for commit:
  modified:   styles.css
```

### 4. Restore all files with dot

```bash
$ git restore .
```
