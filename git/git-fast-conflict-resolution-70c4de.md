# Git: Fast Conflict Resolution

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, merge-conflicts, checkout, conflict-resolution, merge, theirs, ours

## Steps

### 1. Accept all changes from one side

```bash
$ git checkout --theirs .
Updated 3 paths from the index
```

### 2. --ours keeps your changes

```bash
$ git checkout --ours src/
Updated 2 paths from the index
```

### 3. --theirs takes incoming changes

```bash
$ git checkout --theirs package.json
Updated 1 path from the index
```

### 4. Skip manual conflict editing

```bash
$ git add .
$ git commit
[main a1b2c3d] Merge conflicts resolved
```
