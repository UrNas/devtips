# Git: Save Work & Switch Projects Fast

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, stash, branch, workflow, productivity, version-control, development

## Steps

### 1. Working on feature but need to fix urgent bug?

```bash
$ git status
modified:   src/app.js
modified:   src/styles.css
```

### 2. Stash work with descriptive message

```bash
$ git stash push -m "login feature"
Saved working directory and index state
WIP on feature: ab12cd3 login feature
```

### 3. Switch to main for urgent fix

```bash
$ git checkout main
Switched to branch 'main'
```

### 4. After fix, return to feature work

```bash
$ git checkout feature
Switched to branch 'feature'
```

### 5. Restore your work instantly

```bash
$ git stash pop
On branch feature
Changes not staged for commit:
  modified:   src/app.js
```

### 6. Never lose work switching branches again!

```bash
$ git status
modified:   src/app.js
```
