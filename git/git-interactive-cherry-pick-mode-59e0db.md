# Git: Interactive Cherry-Pick Mode

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, cherry-pick, interactive, git-workflow, staging, commit-control, version-control, git-advanced

## Steps

### 1. Cherry-pick with full control over commits

```bash
$ git cherry-pick -n abc123d
Automatic cherry-pick failed.
After resolving conflicts, use git commit
```

### 2. The -n flag stages without committing

```bash
$ git status
Changes to be committed:
  modified:   src/app.js
  new file:   src/utils.js
```

### 3. Edit staged files before committing

```bash
$ git add src/app.js
$ git commit -m "Custom cherry-pick"
[main 4e5f6g7] Custom cherry-pick
2 files changed, 5 insertions(+)
```

### 4. Perfect control over each cherry-pick

```bash
$ git log --oneline -1
4e5f6g7 Custom cherry-pick
```
