# Git: Resolve Conflicts with Patience

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, merge-strategies, patience, histogram, git-config, algorithm, refactoring

## Steps

### 1. Use patience algorithm for cleaner diffs

```bash
$ git merge -X patience feature-branch
Auto-merging src/components.js
Merge made by the 'recursive' strategy.
```

### 2. Or use histogram for moved code blocks

```bash
$ git merge -X histogram feature-branch
Auto-merging src/refactored.js
Merge made by the 'recursive' strategy.
```

### 3. Set as default merge strategy

```bash
$ git config merge.default patience
Configuration updated
```

### 4. Better conflict detection for refactors

```bash
$ git status
On branch main
nothing to commit, working tree clean
```
