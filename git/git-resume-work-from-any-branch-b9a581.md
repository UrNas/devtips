# Git: Resume Work From Any Branch

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, stash, apply, branch, resume, workflow, cross-branch

## Steps

### 1. Forgot which branch has your stashed work?

```bash
$ git stash list
stash@{0}: WIP on feature: login work
stash@{1}: WIP on main: bug fix
```

### 2. Apply any stash to current branch

```bash
$ git stash apply stash@{0}
On branch different-feature
Changes not staged for commit:
```

### 3. Stash stays in list after apply

```bash
$ git stash list
stash@{0}: WIP on feature: login work
stash@{1}: WIP on main: bug fix
```

### 4. Delete stash when you're done with it

```bash
$ git stash drop stash@{0}
Dropped stash@{0}
```

### 5. Work continues anywhere you need it!

```bash
$ git status --short
M  src/app.js
```
