# Git: Quick Stash Without Commit Message

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, stash, quick, fast, untracked, workflow, productivity

## Steps

### 1. Need to stash super fast?

```bash
$ git status --short
M  src/app.js
?? temp.txt
```

### 2. Single command stashes tracked changes

```bash
$ git stash
Saved working directory and index state
WIP on main: abc123 latest commit
```

### 3. Include untracked files too

```bash
$ git stash -u
Saved working directory and index state
```

### 4. Restore with simple pop command

```bash
$ git stash pop
On branch main
Changes not staged for commit:
```

### 5. Fastest way to save and switch!

```bash
$ git log --oneline -1
abc123 latest commit
```
