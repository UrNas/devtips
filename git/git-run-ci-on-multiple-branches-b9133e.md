# Git: Run CI on Multiple Branches

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, worktree, background, parallel, ci, testing, workflow

## Steps

### 1. Set up parallel worktrees

```bash
$ git worktree add ../main main
Preparing worktree (checking out 'main')
HEAD is now at abc1234
$ git worktree add ../dev develop
Preparing worktree (checking out 'develop')
HEAD is now at def5678
```

### 2. Run tests on main in background

```bash
$ cd ../main && npm test &
[1] 12345
Running tests in background...
```

### 3. Run tests on develop simultaneously

```bash
$ cd ../dev && npm test
✓ Tests completed
14 passing, 0 failing
```

### 4. Check background job status

```bash
$ jobs
[1]+  Done    npm test
```
