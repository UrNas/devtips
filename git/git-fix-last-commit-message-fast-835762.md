# Git: Fix Last Commit Message Fast

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, amend, commit, message, fix, typo, last, rewrite

## Steps

### 1. Typo in commit message? Fix it! ✏️

```bash
$ git log --oneline -1
a1b2c3d Fix logn form validation
```

### 2. Amend rewrites the last commit

```bash
$ git commit --amend -m 'Fix login form'
[main b7c8d9e] Fix login form validation
 Date: Mon Dec 4 14:30:15 2023 -0800
```

### 3. Perfect! Message is fixed

```bash
$ git log --oneline -1
b7c8d9e Fix login form validation
```

### 4. Forgot files? Add them first

```bash
$ git add forgot.js && git commit --amend
[main c9d0e1f] Fix login form validation

```
