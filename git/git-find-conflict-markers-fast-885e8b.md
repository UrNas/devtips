# Git: Find Conflict Markers Fast

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, merge-conflicts, grep, vscode, git-mergetool, search, conflict-markers

## Steps

### 1. Search all conflict markers at once

```bash
$ grep -r "<<<<<<< " .
./src/app.js:12:<<<<<<< HEAD
./src/utils.js:45:<<<<<<< HEAD
```

### 2. Count total conflicts remaining

```bash
$ grep -r "<<<<<<< " . | wc -l
2
```

### 3. Jump to next conflict in VS Code

```bash
$ code --goto src/app.js:12
Opening src/app.js at line 12...
```

### 4. Or use git's built-in conflict tool

```bash
$ git mergetool src/app.js
Merging:
src/app.js
```
