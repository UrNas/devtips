# Git: Accept All Changes From One Side

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, merge-conflicts, git-checkout, ours, theirs, resolution, bulk-resolve

## Steps

### 1. Take all changes from current branch

```bash
$ git checkout --ours src/app.js
Updated 1 path from the index
```

### 2. Or take all changes from incoming branch

```bash
$ git checkout --theirs src/utils.js
Updated 1 path from the index
```

### 3. Apply to all conflicted files at once

```bash
$ git checkout --theirs .
Updated 3 paths from the index
```

### 4. Mark as resolved and commit

```bash
$ git add . && git commit
Merge branch 'feature' into main
```
