# Git: 5 Commit Message Mistakes

**Category:** `git`  ·  **Format:** `shorts`

**Tags:** git, commit-mistakes, git-history, clean-commits, professional-workflow, git-conventions, code-quality

## Steps

### 1. These mistakes ruin your git history

```bash
$ git log --oneline -3
abc123d Fixed bug.
def456e WIP: working on feature
ghi789f asdf
```

### 2. 1. Don't end with periods

```bash
# Wrong
Fix login bug.

# Right
Fix login bug
```

### 3. 2. Avoid WIP commits in main

```bash
# Wrong
WIP: still working

# Right
Add initial user model structure
```

### 4. 3. Never use random characters

```bash
# Wrong
asdf, temp, aaa

# Right
Fix header alignment issue
```

### 5. 4. Don't start with lowercase

```bash
# Wrong
fix the bug

# Right
Fix authentication timeout
```

### 6. Clean history = happy developers

```bash
$ git log --oneline -3
x1y2z3a Fix authentication timeout
b4c5d6e Add user profile validation
f7g8h9i Update API error handling
```
